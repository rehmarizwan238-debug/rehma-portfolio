# 🌐 Rehma Rizwan - Frontend React Portfolio

A modern, responsive, and interactive personal portfolio website built with React.js to showcase my frontend development skills, projects, services, and professional journey.

## 🚀 Live Demo

https://rehma-portfolio.vercel.app

## 📂 GitHub Repository

https://github.com/rehmarizwan238-debug/rehma-portfolio

---

## 📌 Project Overview

This portfolio website was developed to showcase my skills and experience as a Frontend React Developer.

The website focuses on:

- Modern UI/UX design
- Responsive layouts
- Clean component-based architecture
- Interactive user experience
- Smooth animations
- Accessibility-friendly elements

---

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Modern Hero section
- About Me section
- Professional Skills section with progress bars
- Services section
- Featured Projects section
- Project search functionality
- Testimonials section
- Contact form with EmailJS integration
- Success and error messages
- Dark/Light mode toggle
- Smooth scrolling navigation
- Active navigation section highlighting
- AOS scroll animations
- Hover effects and interactive UI elements
- Scroll-to-top button
- Resume download functionality
- GitHub and LinkedIn integration

---

- JavaScript (ES6+)
- HTML5

- React Icons

- EmailJS
- Git
